# Runtime APIs

This page outlines the available Cloudflare Workers runtime APIs which Dart Edge provides. The APIs attempt to match
the underlying JavaScript APIs as closely as possible, however provide a "Dartified" API where
it makes sense.

<Warning>
  Implemented APIs are not guaranteed to be stable and may change in future
  releases (this is experimental!).
</Warning>

| Status | Description                                                    |
| :----: | -------------------------------------------------------------- |
|   游릭   | API is implemented (but not fully tested whilst experimental). |
|   游   | API is partially implemented.                                  |
|   游댮   | API is not implemented.                                        |
